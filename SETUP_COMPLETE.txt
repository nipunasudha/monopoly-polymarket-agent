╔════════════════════════════════════════════════════════════════════════════╗
║                          QUICK COMMANDS                                    ║
╚════════════════════════════════════════════════════════════════════════════╝

Activate Environment:
  $ source .venv/bin/activate && export PYTHONPATH="."

Test Installation:
  $ python -c "from agents.connectors.news import News; print('✓')"

Get Market Data (read-only, requires OPENAI_API_KEY):
  $ python scripts/python/cli.py get-all-markets --limit 5

Search News:
  $ python scripts/python/cli.py get-relevant-news "bitcoin"

Ask Superforecaster:
  $ python scripts/python/cli.py ask-superforecaster "Bitcoin ETF" "Pass?" "yes"

Run API Server:
  $ python scripts/python/server.py

Run Trading (requires POLYGON_WALLET_PRIVATE_KEY):
  $ python scripts/python/cli.py run-autonomous-trader

Build Docker Image:
  $ docker build -t polymarket-agents:latest .

Run in Podman:
  $ podman run -it -v $(pwd):/home -v $(pwd)/.env:/home/.env \
    polymarket-agents:latest python scripts/python/cli.py get-all-markets
